Basic

입력받은 string을 기반으로 생성되는 기본 Letter Avatar

Size

입력받은 size에 따라 크기 변경 가능 (1~100, px단위)

{{sizes[i]}}px

Radius

입력받은 size에 따라 둥글기 변경 가능 (1이상, px단위)

radius를 입력하지 않으면 50%로 설정
0 입력 시 50%로 들어가기 때문에 [radius]는 1이상 입력
{{_radius[i]}}px

Circle Border Style

커스텀 스타일링 클래스

[class]="'border-inner'" 클래스 추가

Max Text 2

2글자 까지 표시 (3글자 부터는 따로 작업 필요)
