Basic

기본 입력용 input. 가로로 정렬할 경우 클래스에 'flex' 추가

label과 input을 가로로 정렬 시 .form-group.flex로 묶기

Size

form-group으로 묶으면 label에 클래스를 주지 않아도, form-control의 크기에 따라 사이즈 자동 변환

[group 순서] div.form-group > label + input.form-control
label의 고정 width는 .wd-#{$size}클래스 추가 (10단위 px)

Clearable

입력받은 keyword가 있을 경우 클리어가 가능한 input
div.form-box로 묶으면 안에 있는 form-control의 size에 따라 clear-btn 사이즈 자동 변환
[group 순서] div.form-group > div.form-box > input.form-control + span.clear-btn

clear-btn과 input을 반드시 .form-box로 묶어야 함
close

Icon

아이콘이 있는 input
div.form-box로 묶으면 안에 있는 form-control의 size에 따라 icon 사이즈 자동 변환
[group 순서] div.form-group > div.form-box.icon-left.search > input.form-control

Input에 아이콘을 추가하려면 반드시 input을 form-box로 묶어줘야함 (input에는 after, before 추가가 불가능)
  • [아이콘 위치 지정] 아이콘의 위치에 따라 'icon-left' | 'icon-right' 클래스를 form-box에 추가
  • [아이콘 설정] 아이콘 모양을 $input-icons에 추가 후 form-box에 해당 클래스 추가
  • [로딩] form-box에 [class.loading]="로딩변수", [class.icon-left]="로딩변수" 추가
    * 로딩의 경우 icon-left (icon-right)도 로딩변수로 제어해야함
'stop'입력 시 로딩 멈춤

Validation

필수 입력 input validation check
div.form-box로 묶으면 안에 있는 form-control의 size에 따라 valid-feedback 사이즈 자동 변환
[group 순서] div.form-group > div.form-box > input.form-control + span.valid-feedback(invalid-feedback)

valid 사용 시 form-group에 flex를 붙이지 말 것
'success'입력 시 is-valid 클래스 추가 (clear-btn 조건과 valid 조건이 충돌하지 않게 주의)
close 검증 완료
입력 없을 시 is-invalid 클래스 추가
close 필수 입력

Group

다수의 input group의 layout

div.d-flex.flex-column으로 묶은 경우
