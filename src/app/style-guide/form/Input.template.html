Basic

기본 입력용 input. 가로로 정렬할 경우 클래스에 'flex' 추가
간격 조정은 mb-1, me-2 등으로 조절

Size

form-group으로 묶으면 label에 클래스를 주지 않아도, form-control의 크기에 따라 사이즈 자동 변환

[group 순서] div.form-group > label + input.form-control
label과 input을 반드시 form-group으로 묶어야 함
label의 width 지정시 wd-N클래스 추가 (10단위 px)

Clearable

입력받은 keyword가 있을 경우 클리어가 가능한 input
div.form-group으로 묶으면 안에 있는 form-control의 size에 따라 clear-btn 사이즈 자동 변환
[group 순서] div.form-group > input.form-control + span.clear-btn

clear-btn과 input을 반드시 form-group으로 묶어야 함
close

Validation

필수 입력 input validation check
div.form-group으로 묶으면 안에 있는 form-control의 size에 따라 valid-feedback 사이즈 자동 변환
[group 순서] div.form-group > input.form-control + span.valid-feedback(invalid-feedback)

'success'입력 시 is-valid 클래스 추가 (clear-btn 조건과 valid 조건이 충돌하지 않게 주의)
form-group.flex 일 경우, valid-feedback(invalid-feedback)에 padding-left를 추가하기 위해 'pl-N' ( 2~80 짝수) 클래스 추가
pl-N은 label의 wd-(x)값에 따라 적당히 (x) + margin값으로 계산
close 검증 완료
입력 없을 시 is-invalid 클래스 추가
close 필수 입력

Group

다수의 input group의 layout

margin-bottom 추가 시 클래스에 mb-1, mb-2, mb-3 추가
label과 input을 가로로 정렬 시 div.form-group에 flex 클래스 추가
form-group을 가로로 정렬 시 div.form-grid로 묶기
form-group.flex를 가로로 정렬 시 div.form-grid로 묶기
form-grid에서 각 form-group의 가로 사이즈 설정 시 div.form-group에 col-md-N(12,6,4,3,2,1) 클래스를 추가
* md뒤 숫자 N은 width: N/12를 뜻함
