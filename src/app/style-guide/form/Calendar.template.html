Basic

기본 날짜 선택 Date Picker로 기본 사이즈는 form-control-sm

<date-picker> 옵션 설정 (입력 안 했을 경우 기본값)

옵션 없이 <date-picker> 사용 가능 (단일 날짜 picker)

  • [size] 'xs' | 'sm' | 'default' (input.form-control size를 결정)
  • [dateInputFormat] 'YYYY-MM'-DD' | 'YYYY.MM.DD H:ss' | 'YYYY/MM/DD h:ss a' | 나머지 포맷 moment.js 참고
  • [placeholder] 입력 안할 경우 dateInputFormat | string 입력
  • [locale] 'kr' | 'en' | 'cn'
  • [minDate], [maxDate] Date format 변수
  • [customTodayClass] 'ngx-today' | '' (ngx-today 입력 시 오늘 날짜 스타일링)
  • [dateTooltipTexts] [ { date: Today, tooltipText: '오늘' } ] | tooltipDates: DatepickerDateTooltipText[] (툴팁 변수에 date, tooltipText 지정)
  • [withTimepicker] true | false
  • [showWeekNumbers] true | false
calendar와 input의 가로 사이즈를 맞추려면 date-picker에 'wd-220' 클래스 추가
선택한 날짜 : {{selectedDate | json}}

Range Datepicker

날짜 범위 선택 Range Picker로 기본 사이즈는 form-control-sm

range-picker의 경우 [dateInputFormat] 대신 [rangeInputFormat]을 입력해야함
<date-range-picker> 옵션 설정 (기본값)

<date-range-picker> 사용시 [rangeInpitFormat] 입력 필수 (날짜 범위 선택)

  • [size] 'xs' | 'sm' | 'default' (input.form-control size를 결정)
  • [rangeInputFormat] 'YYYY-MM'-DD' | 'MM/DD/YYYY' | 'MMMM Do YYYYY, h:mm:ss a'
  • [placeholder] 입력 안할 경우 rangeInputFormat과 동일 | string 입력
  • [locale] 'kr' | 'en' | 'cn'
  • [minDate], [maxDate] Date format 변수
  • [preNext] true | false (선택한 범위 만큼 이동하는 버튼)
  • [customTodayClass] 'ngx-today' | '' (ngx-today 입력 시 오늘 날짜 스타일링)
  • [dateTooltipTexts] [ { date: Today, tooltipText: '오늘' } ] | tooltipDates: DatepickerDateTooltipText[] (툴팁 변수에 date, tooltipText 지정)
  • [withTimepicker] true | false
  • [ranges] true 입력 시 기본값[최근 1일, 최근 일주일, 최근 1개월, 최근 3개월, 최근 6개월, 최근 1년] | ranges: IRange[] (자동 선택 범위)
  • [showWeekNumbers] true | false
calendar와 input의 가로 사이즈를 맞추려면 date-range-picker에 'wd-440' 클래스 추가
선택한 날짜 : {{selectedRange | json}}

Time Picker

시간 선택 Time Picker

<timepicker> 옵션 설정 (기본값)
  • size timepicker 사용 시 무조건 time-base, time-sm, time-xs 중 1개 클래스 추가
  • [disabled] true | false
  • [hourStep] 1 | number (스크롤 스텝)
  • [minStep] 5 | number (스크롤 스텝)
  • [showMeridian] true (12H) | false (24H)
  • [meridians] ['AM', 'PM'] | ['오전', '오후']
  • [min], [max] Date 변수 선언 후 setHours(), setMinutes()로 설정
timepicker에 클래스를 입력하지 않을 경우, 커스텀 스타일이 적용되지 않음
