Basic

기본 날짜 선택 Date Picker
[minDate], [maxDate] // Date format 변수
[bsConfig.outsideClick]: true | false // 바깥 클릭 시 닫힘
[bsConfig.dateInputFormat]: 'YYYY-MM'-DD' | 'MM/DD/YYYY' | 'MMMM Do YYYYY, h:mm:ss a'
[bsConfig.showWeekNumbers]: true | false // week 표시

Clear

선택된 날짜 제거 clear button
clearPosition: 'left' | 'center' | 'right'

Range Datepicker

날짜 범위 선택
[bsConfig.rangeInputFormat]: 'YYYY-MM'-DD' | 'MM/DD/YYYY' | 'MMMM Do YYYYY, h:mm:ss a'
[bsConfig.ranges]: IRange[] //범위 자동 선택

Date Time picker

날짜와 시간 선택
[bsConfig.rangeInputFormat]: 'YYYY-MM'-DD' | 'MM/DD/YYYY' | 'MMMM Do YYYYY, h:mm:ss a'

Picker Position

Picker 위치 설정
[bsConfig.adaptivePosition]: true | false //스크롤 위치에 따라 자동 포지션
placement: 'bottom' | 'top' | 'right' | 'left' //picker 위치 설정s

Custom Today

오늘 날짜 Custom Class
[bsConfig.customTodayClass]: 'ngx-today' //오늘 날짜에 class 추가

Tooltip For Date

특정 날짜에 tooltip 추가
[bsConfig.dateTooltipTexts]: DatepickerDateTooltipText[] //툴팁 변수에 date, tooltipText 지정
