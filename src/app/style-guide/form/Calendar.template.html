기본 날짜 선택 Date Picker

date-picker 옵션 설정 (입력 안 했을 경우 기본값)

옵션 없이 date-picker 사용 가능 (단일 날짜 picker)

  • [size] 'sm' | 'xs' | 'default' (input.form-control size를 결정)
  • [placeholder] dateInputFormat과 동일 | string 입력
  • [locale] 'kr' | 'en' | 'cn'
  • [minDate], [maxDate] Date format 변수
  • [dateInputFormat] 'YYYY-MM'-DD' | 'MM/DD/YYYY' | 'MMMM Do YYYYY, h:mm:ss a'
  • [outsideClick] true | false (영역 바깥 클릭 시 닫힘)
  • [showWeekNumbers] true | false
  • [returnFocusToInput] true | false (날짜 선택 후 input에 focus)
  • [placement] (adaptivePosition:true) | 'bottom' | 'top' | 'left' | 'right' (캘린더 위치, 입력 안 할 경우 adaptivePosition:true)
  • [customTodayClass] 'ngx-today' | '' (ngx-today 입력 시 오늘 날짜 스타일링)
  • [dateTooltipTexts] [ { date: Today, tooltipText: '오늘' } ] | tooltipDates: DatepickerDateTooltipText[] (툴팁 변수에 date, tooltipText 지정)
calendar와 input의 가로 사이즈를 맞추려면 date-picker에 'wd-280' 클래스 추가

Range Datepicker

날짜 범위 선택 Range Picker (기본 옵션은 Date Picker와 동일)

range-picker의 경우 [dateInputFormat] 대신 [rangeInputFormat]을 필수로 입력해야함
range-picker 옵션 설정 (기본값)

range-picker 사용시 [rangeInpitFormat] 입력 필수 (날짜 범위 선택)

  • [rangeInputFormat] 'YYYY-MM'-DD' | 'MM/DD/YYYY' | 'MMMM Do YYYYY, h:mm:ss a'
  • [ranges] [최근 일주일, 최근 1개월, 최근 3개월, 최근 6개월, 최근 1년] | ranges: IRange[] (자동 선택 범위)
