Basic

기본 Knob

Min/Max

범위는 기본값이 각각 0과 100인 [min] 및 [max] 속성으로 구성됩니다.

Step

각 이동의 크기는 [step] 속성으로 정의됩니다.

Value Template

Label은 자리 표시자로 60을 갖는 [valueTemplate] 속성을 사용하여 사용자 정의할 수 있는 문자열 템플릿입니다.

Stroke Width

테두리 크기는 [strokeWidth] 속성을 사용하여 픽셀 단위로 지정됩니다.

Size

손잡이의 직경은 [size] 속성을 사용하여 픽셀 단위로 정의됩니다.

Color

색상은 [textColor], [rangeColor], [valueColor] 속성을 사용하여 사용자 정의됩니다.

ReadOnly

읽기 전용이 있으면 값을 편집할 수 없습니다. [readonly]

Disabled

비활성화된 경우 Knob와 상호 작용할 수 없음을 나타내는 시각적 힌트가 적용됩니다. [disabled]
