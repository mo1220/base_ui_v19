ag-grid에 cellRendrer 설정

template에 ag-grid-angular에 [components]="frameworkComponents"를 추가한 후 cellRenderer 설정 가능

  • CellLoadingComponent skeleton loading에 사용
    cellRendererParams: { loading: this.loading }
  • ButtonRenderComponent 아이콘 버튼 추가 시 사용할 수 있다.(배열로 받아야 합니다!!)
    배열로 여러개를 받아 Action Group을 생성할 때 활용할 수 있고, 이벤트는 onClick 하나로 받아 actionType으로 구분하여 사용
    cellRendererParams: {
    btns: [
    {
    icon: 'settings',
    tooltip: '수정',
    actionType: 'modify',
    },
    ...
    onClick: (d:any) => this.onConfigClick(d)
    }
  • ChipRenderComponent 키워드 형태로 추가 시 사용 가능
    cellRendererParams는 필요 없지만 데이터가 array<string> 형태여야함
ColGroupDef

columnDefs: (ColDef | ColGroupDef)[] = [];로 선언

  • marryChildren 부모에 marryChildren: true를 반드시 추가하셔야 합니다.
  • headerClass 각 헤더 그룹의 스타일을 구분해주기 위해 headerClass: 'className'을 추가할 수 있습니다. css 예시: .col-group-grid
